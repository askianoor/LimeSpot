The Assignment

    • Do not use any 3rd-party library.
    • Do not change index.html.
    • Do not change index.js.
    • Create a dialog component with className of 'ls-dialog'.
    • Create a popover component with className of 'ls-popover'.
    • You can write your JavaScript code in main.js and your CSS code in index.css

Please try completing the assignment without removing and/or changing the existing code. You can add new code.

  • Inside the page, there is a button. On click, a dialog should be shown with the following criteria:
    • It should be opened in the center of screen
    • It should have a backdrop with a dark color: rgba(0, 0, 0, 0.32)
    • The document body intentionally forces a vertical scroll. You should handle the scroll when the dialog is opened. When the dialog is opened, the content behind it should not be scrollable.
    • When the user clicks outside of the dialog, it should be automatically dismissed and closed but when clicked inside, it should remain open.
  • Inside the dialog there is a button. On click, a popover should be shown with the following criteria:
    • It has no dark backdrop.
    • It should be dismissed when clicked outside of it.
    • It has a triangle pointer on top implemented using CSS only.
    • It has an input where user can enter a value.
    • It has two buttons: "Close" which dismisses the popover and "OK" which accepts the value and shows it inside a new dialog opened on top of everything.

Assignment submission

While there's no limit, to be considerate of your time, we ask you not to spend more than 4-5 hours working on the project. Whatever you can accomplish in that time would be enough for us to discuss in a code review session.

The goal of this assignment is to evaluate how you would solve a problem, how you would architect and design it and how clean your code is. Completeness is not an objective. You can demonstrate the above with a project that is not finished yet but is a good / solid start of some greatness!

When you're happy with what you have, send back a ZIP archive of your work by replying to assignment email and we'll get back to you withing 24-48 hours for next steps!
